Welcome to pbc_fix!

This little Python utility correct Molecular Dynamics trajectories in which the systems simulated deforms and expands beyond the dimensions of the simulation box.

$ python pbc_fix.py -f file.gro -x file.xtc

At present it only works for rectangular simulation boxes. A list of all commands is available with:

$ python pbc_fix.py --help
