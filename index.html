Pbc fix

Fix problematic crossing of box boundaries in MD trajectories.

View on GitHub Download .zip Download .tar.gz

Welcome to pbc_fix!

This little Python utility correct Molecular Dynamics trajectories in which the systems simulated deforms and expands beyond the dimensions of the simulation box. At present it only works for rectangular simulation boxes.

If you find it useful please cite it:

Usage

$ python pbc_fix.py -f file.gro -x file.xtc
$ python pbc_fix.py --help
